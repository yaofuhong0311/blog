## 这是什么

skill-hub 是一个管理本地 Claude Code skill 的桌面应用：把散落在各处的 skill 集中到一个界面里，可以浏览每个 skill 的说明、启用/停用，不用再手动翻目录改文件。

## 为什么做它

Claude Code 的 skill 多了以后，哪些装了、哪些生效、每个是干嘛的，很难一眼看清。skill-hub 把这些信息可视化，管理成本降下来。
